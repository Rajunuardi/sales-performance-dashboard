# Sales Performance Dashboard (Excel)

## Objective
Membangun dashboard penjualan berbasis Excel
untuk membantu monitoring performa penjualan,
tren bulanan, dan kontribusi region.

## Tools Used
- Microsoft Excel

## Dataset
- Dummy sales dataset
- Format: Excel (.xlsx)

## Process
- Data cleaning
- Perhitungan KPI menggunakan Excel
- Pembuatan dashboard interaktif

## Key Insights
- Performa penjualan per region dapat dipantau dengan cepat
- Tren penjualan membantu evaluasi bulanan
